---
cluster list는 Cocktail DashBoard 또는 Cocktail Online에 등록된 Cocktail Cluster를 정보를 표시한다. 

---

### Prerequisite
Cocktail Cluster 설치 상태여야 한다.  
`create` command 

Cocktail DashBoard 또는 Cocktail Online에 로그인된 상태여야 한다.  
`login` command 

----
### Usage

* `cube cluster-list`


### Sample
```
# cube cluster-list
```
