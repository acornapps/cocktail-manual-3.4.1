서버에서 Json 형태로 저장하여 워크로드의 네트워크 설정으로 사용 할 수 있다.
네트워크 메뉴는 애드온 목록에 multi-nic가 설치되 있어야 설정 가능하다.

### a\) 좌측 메인메뉴 → 서비스 → 서비스 카드 선택 → 설정 정보 → 네트워크 클릭한다.
