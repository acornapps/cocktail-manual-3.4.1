---
cluster list は Cocktail DashBoard または Cocktail Online に登録された Cocktail Cluster の情報を表示する。 

---

### Prerequisite
Cocktail Cluster がインストールされてなければならない。  
`create` command 

Cocktail DashBoard または Cocktail Online にログインされてなければならない。  
`login` command 

----
### Usage

* `cube cluster-list`


### Sample
```
# cube cluster-list
```
