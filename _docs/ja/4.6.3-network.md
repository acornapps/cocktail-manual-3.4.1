サーバーから Json 形式で保存して、ワークロードのネットワーク設定として使用することができる。
ネットワークメニューは、Add-on リストに multi-nic がインストールされていれば設定可能である。

### a\) 左側メインメニュー → サービス → サービスカード選択 → 設定情報 → ネットワークをクリックする。
