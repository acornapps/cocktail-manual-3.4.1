---
logout はインストールされた Cluster の Cocktail DashBoard またｈ Cocktail Online サービスからログアウトする。

---

### Prerequisite
Cocktail Cluster インストールされてなければならない。  
`create` command 

Cocktail DashBoard または Cocktail Online にログインされてなければならない。  
`login` command 

----
### Usage

* `cube logout`



### Sample
```
# cube logout
```
