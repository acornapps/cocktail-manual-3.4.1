クラスターを構成する各 node は cache memory が
徐々に増加するようになっている。

定期的に各 node で以下のコマンドを実行し、cache memory を初期化できる。

```
# sync; echo 3 > /proc/sys/vm/drop_caches 또는
# sync; sysctl -w vm.drop_caches=3

```
